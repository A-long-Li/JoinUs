食用指南
- 打开`configs/cnf.ini`配置文件进行配置
- 进入到main.go所在文件夹下运行`go build -o (name.exe) main.go`来打包go项目
- 在和`configs`同级文件夹下运行以下命令`.\{name}.exe configs/cnf.ini`来运行
